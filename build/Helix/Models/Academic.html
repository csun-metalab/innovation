class Academic extends Model

Class Academic

Note: This class is interesting to say the least. This class represents a "model" that's only represented in the pivot table. So, that's why you might see that the table is expertise_entity. We're doing this because the model is just a mapping to the research interest tree. So, throughout the application, you're going to see string manipulation that appends ":academic to table", and whenever there is need to modify the pivot or research table, you'll see the removal of that string.
