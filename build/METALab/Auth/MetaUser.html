MetaUser

class MetaUser extends Model implements Authenticatable, MetaAuthenticatableContract

Base class used with the META+Lab authentication mechanism. This can be used on its own or subclassed for a specific application. If used on its own it expects a table called "users" with a primary key of "user_id". The constructor can also be used to provide an instance with modified table and primary key names as well.
